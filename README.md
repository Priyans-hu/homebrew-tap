# Homebrew Tap

Homebrew formulae and casks for tools by [Priyanshu Garg](https://github.com/Priyans-hu).

## Usage

```bash
brew tap Priyans-hu/tap
```

## Available Formulae

| Formula | Description | Install |
|---------|-------------|---------|
| [argus](https://github.com/Priyans-hu/argus) | Auto-generate AI context files from your codebase | `brew install Priyans-hu/tap/argus` |
| [sreq](https://github.com/Priyans-hu/sreq) | Service-aware API client with automatic credential resolution | `brew install Priyans-hu/tap/sreq` |

## Available Casks

| Cask | Description | Install |
|------|-------------|---------|
| [tokenmeter](https://github.com/Priyans-hu/tokenmeter) | macOS menu bar app for Claude Code token usage visualization | `brew install --cask Priyans-hu/tap/tokenmeter` |

## Updating

```bash
brew update
brew upgrade Priyans-hu/tap/<formula>
```
